### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
   
🌿 Self-Care Assessment Tool

This project is an interactive web application developed using Streamlit. It is designed to help users reflect on their current self-care practices across five domains:

Physical Self-Care
Psychological Self-Care
Emotional Self-Care
Spiritual Self-Care
Workplace / Professional Self-Care

Users rate each statement on a 5-point scale, and the application provides a summary of their self-care levels along with opportunities for reflection.

📚 Source of Assessment

The self-care assessment items used in this application are adapted from the following resource:

Articulate Rise course:
Self-Care Assessment Tool

This resource was created using Articulate Rise, an e-learning authoring tool that allows developers to build interactive learning content and assessments .

🎯 Purpose

This tool aims to:

Promote self-awareness and reflection
Support holistic wellbeing
Encourage users to identify areas for growth in self-care
Align with holistic approaches to wellbeing in education and professional practice

Self-assessment is an important reflective process that helps individuals understand their strengths and areas for improvement and supports self-directed learning .

⚠️ Disclaimer

This tool is intended for educational and reflective purposes only.
It is not a diagnostic or clinical assessment.
